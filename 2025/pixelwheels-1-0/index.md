## What's new?

There aren't a lot of changes in this version, which is a good sign considering that the previous version was a release candidate. Let's have a look at them.

### HUD improvements

I improved the "pie buttons" that are used to control the game on mobile:

- You can now make them bigger if necessary;
- They are easier to use thanks to their hit zones now extending as far as possible;
- I fixed a bug in the way the HUD images were generated that caused them to end up flat. Now they once again have a little edge, which looks neat I think.

![Side buttons HUD, with depth back][hud]

[hud]: hud.png

### Indonesian translation

Pixel Wheels learned to speak Indonesian, thanks to Neko Nekowazarashi. It always makes me happy when new translations are added because it makes the game usable by more people.
